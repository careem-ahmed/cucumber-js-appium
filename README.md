# cucumber-js-selenium

pre-requisite (have to be installed and added to path)
	-	node
	-	npm
	-	chrome driver
	-	appium
	-	android studio 

to install dependencies
```

npm install
```

run on pc

```
./node_modules/.bin/grunt 
```

run on android

```
./node_modules/.bin/grunt  android
```

run on ios

```
./node_modules/.bin/grunt  ios
```
